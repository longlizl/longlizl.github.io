GIT安装与使用

by · 2017年01月20日 · 185 Words · ~1min reading time | Improve on

第一次接触git分布式版本控制系统简单快速的功能给吸引住了！废话不多说了让我们来看下 它在windows下安装。

在Windows上安装Git

  • 请点击git下载安装包或去官网下载安装包。
  • 安装完成后，在程序里找到“Git”->“Git Bash”，看到一个命令行窗口的东西，就说明Git安装成功！

  • 安装完毕后，还需要指定你的用户名和email地址。输入以下命令：
    git config –global user.name “name”
    git config –global user.email “email address”
  • 使用 git config -l 或 –list 查看全局配置信息。
